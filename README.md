# Система бронирования боксов автосервиса

Веб-приложение для онлайн-бронирования ремонтных боксов автосервиса самообслуживания.

## Функции

- Бронирование боксов
- Авторизация администратора
- Просмотр бронирований
- Удаление бронирований
- Календарь занятости
- Защита от двойного бронирования

## Технологии

- Python
- Flask
- SQLite
- HTML5
- Bootstrap 5
- JavaScript

## Запуск

```bash
## Данные администратора

Логин: admin

Пароль: admin

pip install -r requirements.txt
python app.py
```
