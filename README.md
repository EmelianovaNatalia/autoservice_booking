# autoservice_booking

## Описание
Веб-приложение для онлайн-бронирования автосервиса самообслуживания.

## Цель проекта
Разработать систему, которая позволяет пользователям бронировать ремонтные боксы онлайн.

## Технологии
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Функционал (будет реализован)
- просмотр боксов
- бронирование времени
- управление заявками
- админ-панель
